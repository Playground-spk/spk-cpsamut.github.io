การเขียน css มี inline (อยู่ในtag) internal CSS : เขียนภายใน Element HTML external CSS : นำเข้าไฟล์ CSS จากภายนอก
selector มีกี่แบบ ID : การกำหนดชื่อของ element โดยจะมีได้แค่ element เดียวเท่านั้น Class : การกำหนดชื่อของ element โดยสามารถมีได้หลาย element Pseudo element : การกำหนดเฉพาะบางส่วนของ element Pseudo class : การกำหนดเฉพาะบางสถานะของ class Combinators : การกำหนดเงื่อนระหว่าง 2 selector attribute selector : การกำหนดการเลือกชื่อ attribute ใน element
combinator มีกี่แบบ descendant selector (space) : a b = tag b ทั้งหมดที่อยู่ใน a child selector (>) : a > b = tag b ที่เป็นลูกของ a adjacent sibling selector (+) : a + b = tag b ที่อยู่ล่าง a และติดกับ a เท่านั้น general sibling selector (~) : a ~ b = tag b ที่อยู่ล่าง a ทุก tag
